# Conceito Pastoral do Aplicativo

## Ponto de partida
Criar um aplicativo que transforma os eventos (encontros) dos grupos
em uma Agenda Anual Pastoral partilhável.

## Ideia central
Um aplicativo (web + mobile) que apresenta a vida pastoral como:

- 📅 Agenda interativa (por dia, mês, grupo)
- 📘 E-book anual (PDF / EPUB)
- 🔄 Base viva, atualizável ao longo do ano
- ♻️ Modelo reutilizável para os anos seguintes

## Princípio pastoral
Não é apenas um calendário.

É a vida comunitária:
- organizada
- visível
- partilhável

O aplicativo existe para servir a comunhão,
a memória e o planejamento pastoral.
